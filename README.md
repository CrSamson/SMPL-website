# SMPL Innovations Website

Static Vercel-ready refactor for the SMPL Innovations vitrine website.

## Files

- `index.html` contains the main one-page sales site.
- `styles.css` contains the responsive visual system.
- `script.js` handles the mobile menu and dynamic year.
- `vercel.json` redirects old Wix routes to the new sections.
- `privacy.html`, `cookies.html` and `legal.html` replace the Wix legal templates.

## Before launch

- Replace `assets/logo.svg` with the official SMPL logo if a source file is available.
- Confirm the Cal.com slug and update the booking links if needed.
- Replace reference-ready proof cards with approved client testimonials.
- Add real client logos only after permission is confirmed.

## Checks

```bash
npm run test
npm run build
```
